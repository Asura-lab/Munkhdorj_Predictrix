# Predictrix v0.4.5 (APK only)

Энэ хувилбар нь Android APK түгээлтэд төвлөрсөн бөгөөд нэвтрэлт, сессийн тогтвортой байдал, ML сигналын зөв тооцоолол, релиз тохиргооны найдвартай ажиллагааг сайжрууллаа.

## 1) Шинэчлэлтийн гол агуулга
- Auth endpoint-ууд нэмэгдэж, хамгаалалт нь чангарав.
- JWT баталгаажуулалт сайжирч, сессийн удирдлага илүү найдвартай боллоо.
- Push notification инициализаци тогтвортой болж, эхлэлийн warning-ууд буурсан.
- API орчны логик сайжирч, хөгжүүлэлт дээр local API, релиз дээр Fly API-г зөв ашигладаг боллоо.
- Нэвтрэх, профайл, нууц үг сэргээх урсгалууд хоорондоо илүү нийцтэй болсон.

## 2) Засвар, тогтвортой байдал
- Background worker lock-ийн зан төлөвийг засч, давхардсан/уралдаант ажлын эрсдэлийг бууруулсан.
- Auth/session/profile/reset урсгалын алдаа боловсруулах ба төлөв шинэчлэх логик тогтвортой болсон.
- ML хэсэгт model path ачаалалтын алдааг засаж, multiclass probability тооцооллыг зөв болгосон.
- Android релизийн тохиргоо, версийн зөрүүтэй холбоотой асуудлуудыг зассан.

## 3) Техникийн өөрчлөлт
- Сервер талд strict env шалгалт болон runtime flag-уудыг чангатгасан.
- Deploy скрипт env-first secrets зарчимд шилжсэн.
- Signal generator болон GBDT model кодод зам заалт, ангиллын магадлалын засвар хийсэн.
- Mobile талын API service болон Login/ForgotPassword/Profile урсгалын логик нэг стандарт руу нийцсэн.
- Тохиргооны файлуудын релиз барилттай холбоотой засварууд хийгдсэн: tsconfig, app metadata, package тохиргоо, Android build тохиргоо.

## 4) Анхаарах зүйл
- Энэ релиз нь зөвхөн Android APK-д хамаарна.
- Релиз горимд апп Fly API руу автоматаар холбогдоно.
- Хуучин сесс/токентой хэрэглэгч нэвтрэлтийн асуудалтай бол дахин нэвтрэхийг зөвлөж байна.
- Push notification ажиллуулахын тулд мэдэгдлийн зөвшөөрөл идэвхтэй байх шаардлагатай.
- Backend strict env шалгалтаас шалтгаалан шаардлагатай орчны хувьсагч дутуу бол сервис асахгүй байж болно.
